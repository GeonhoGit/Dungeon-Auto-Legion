# ⚔️ Dungeon Auto Legion (던전 오토 레기온)

**Dungeon Auto Legion**은 브라우저에서 설치 없이 가볍게 즐길 수 있는 **로그라이크 오토배틀러(Auto-battler)** 웹 게임입니다. 
다양한 직업을 가진 용병들을 고용하고, 진형을 갖춰 던전의 끝에 도달하세요!

---

## 🌟 핵심 기능 (Key Features)

* **🛡️ 다채로운 시너지 조합:** 전사, 마법사, 궁수, 치유사 등 다양한 직업들을 조합해 강력한 시너지 효과를 발동시키세요. 어떤 조합을 맞추느냐에 따라 전투의 양상이 180도 달라집니다.
* **✨ 유닛 진화 시스템:** 같은 등급의 유닛 3마리를 모으면 다음 등급으로 자동 진화합니다. (최대 신화 및 각성 등급까지 존재합니다!)
* **🗺️ 로그라이크 탐험:** 매번 다르게 생성되는 던전 지도에서 전투, 상점, 휴식, 이벤트 방을 신중하게 선택해 나만의 루트를 개척하세요.
* **⚔️ 자동 전투 & 사령관 스킬:** 필드에 유닛을 배치하면 전투는 자동으로 진행됩니다. 플레이어는 적절한 타이밍에 사령관 스킬(회복, 벼락)을 사용해 전황을 뒤집을 수 있습니다.
* **💎 영구 강화 (메타 프로그레션):** 보스를 처치하고 얻은 보석으로 다음 게임에서도 영구적으로 유지되는 강력한 버프를 제단에서 해금하세요.
* **📊 DPS 미터기:** 매 전투가 끝날 때마다 유닛들의 가한 피해량, 받은 피해량, 회복량 통계를 한눈에 분석할 수 있습니다.

---

## ⌨️ 조작 방법 (Controls)

* **마우스 (좌클릭):** 유닛 배치, UI 상호작용 및 선택
* **Spacebar:** 전투 시작, 결과 대기 스킵, 다음 방으로 빠르게 이동
* **숫자 키 (1~9):** 상점/보상/이벤트 선택 단축키
* **R 키:** 상점 목록 새로고침
* **ESC 키:** 열려있는 각종 도감 및 모달 창 닫기

---

## 🛠️ 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (No Frameworks)
* **Storage:** LocalStorage API (Save/Load)
* **Architecture:** SPA (Single Page Application)

서버나 백엔드, 별도의 라이브러리 없이 **순수 바닐라 자바스크립트**만으로 구현된 프로젝트입니다.
